# Web3 Token & Protocol Compliance Auditor

**Author:** Jefferson Mong’are  
**Role:** Web3 Legal Counsel at MetaLeX Labs | Governance Director at Ychad.Eth (Yearn Finance DAO-aligned BORG)

**First dedicated Web3 Agent Skill** in the Lawvable library.

This skill turns any LLM (Claude, GPT, Gemini, etc.) into a structured, disciplined Web3 legal compliance analyst for token launches, DAOs, DeFi protocols, and smart contracts.

## Features
- Howey / Reves / MiCA / FATF analysis
- Smart contract legal alignment review
- DAO governance & centralization risk assessment
- Professional risk heatmap + memo output
- Strong hallucination guardrails & disclaimers

## How to Use
1. Download or copy `SKILL.md`
2. Paste it as a custom instruction / skill in Claude, ChatGPT, or Gemini
3. Upload whitepaper, tokenomics, Solidity code, or project description
4. Run analysis
