# MiCA Quick Reference

- **Crypto-asset** — Broad category
- **ART (Asset-Referenced Token)** — Stabilised by reference to any assets
- **EMT (E-Money Token)** — Stabilised by reference to one official currency
- **Utility Token** — Intended only for goods/services (narrow exemption)

**Key Obligations:**
- Whitepaper approval
- Issuer authorisation
- Reserve assets & custody requirements
- Marketing & consumer protection rules
- CASP (Crypto-Asset Service Provider) licensing if applicable
